# 🛡️ Insurance Claim Fraud Detection System

A Machine Learning-based Insurance Claim Fraud Detection System developed using Python, Scikit-learn, and Streamlit. This project analyzes insurance claim data, detects fraudulent claims, and provides an interactive dashboard for prediction and visualization.

---

## 📌 Project Overview

Insurance fraud is a major challenge for insurance companies, leading to significant financial losses. This project uses Machine Learning techniques to classify insurance claims as **Fraudulent** or **Genuine** based on customer, policy, vehicle, and claim-related information.

The project includes an interactive **Streamlit Dashboard** where users can enter claim details and predict whether a claim is fraudulent.

---

## 🎯 Objectives

- Detect fraudulent insurance claims using Machine Learning.
- Perform data cleaning and preprocessing.
- Analyze claim patterns using Exploratory Data Analysis (EDA).
- Build and compare multiple Machine Learning models.
- Deploy the best-performing model using Streamlit.

## ▶️ Installation

Clone the repository:



Install dependencies:

